这是一个有着高角色真实度与高自由度共存的AI角色扮演引擎，同时有着大量高级功能，以供DIY特定场景需求。
1. 稳定且通用的轮次系统：所有角色行为都以串行的轮次系统构建，通过轮次顺序判定，角色行为轮次，反应轮次插入以及特殊的结算轮次，引擎实现了灵活多样的角色互动效果。
2. 独立的角色认知：角色的公开物品、私人物品和世界内的个人记忆完全独立，角色在世界上移动地点也不会影响其记忆，彻底解决现有AI角色扮演引擎串戏的问题。
3. 丰富的行为逻辑：物品与技能的命中判定同时结合上下文与数值属性进行智能判断，支持角色进行奖池抽奖、自然语言交易、盗窃或抢夺、移动地点等复杂逻辑，并且AI角色与玩家行为能力对等。
4. 独特的驱力-矛盾系统：角色在低快感情况下会有强驱力满足快感，而高快感情况下会尝试解决地方矛盾。重复满足快感将刺激奖励机制绑定在特定快感模式中（成瘾），而解决矛盾会给角色带来通用货币以自行创造新物品或技能。
5. 合理且随机的世界地图：在随机生成的世界地形上，存在雪山、溪流、湖泊、海洋、城市等多种地形，根据不同的地形和区域地理特征，模型将智能生成特定的区域信息与区域中地点的信息。这创造了一个可信的世界，让角色在其中能够有宏观的认知。地方矛盾的解决将让玩家增加可以前往更远地点的行动点数。
6. 灵活的高级功能：如果觉得现有逻辑不能满足特定的游戏需求，除了可以完全自定义区域、地点、角色、技能、物品、判定逻辑等，还有自定义奖池、数值触发器(beta)、自定义轮次、上下文工程、全局变量(beta)等高级功能等着你。
7. 信息明确的界面：界面以清晰的逻辑拆分为左侧区域信息，右侧轮次信息，下方玩家控制栏，上方游戏选项，正中间故事日志的构成，且在不同设备上可以灵活切换横竖屏显示。还有颜色主题设置哦。
